BD Economics

Overview

NOTE (11/16/2019): Data on this site are no longer being updated.
Follow @bd_econ for updates

This site is a repository for code and data that others may find useful for their own research. The Reports section has lots of data and data visualizations, while the Tools section contains guides for working with economic data using python. My blog is located at http://www.briandew.wordpress.com.

Future plans, some of which are already work in progress, include 1) new guides for working with microdata like the Current Population Survey, American Community Survey, and Survey of Consumer Finance, 2) a trade dynamics dashboard, and 3) a revamped chart book that captures new labor market indicators.
