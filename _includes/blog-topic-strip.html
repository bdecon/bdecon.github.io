{%- comment -%} Compact topic-nav strip for blog pages beyond the index. Lets a reader jump to a different category from /blog/page2+ or from any /blog/category// page without going back to /blog/ first. Pass `current` (the active category name) to highlight that chip, or omit it on non-category pages. Usage: {% include blog-topic-strip.html %} # /blog/ page 2+ {% include blog-topic-strip.html current=page.category %} # category page {%- endcomment -%}
All posts {%- for cat_pair in site.categories %} {%- assign cat_name = cat_pair[0] -%} {%- assign _visible = cat_pair[1] | where_exp: "p", "p.unlisted != true" -%} {%- assign cat_count = _visible | size -%} {%- if cat_count == 0 %}{% continue %}{% endif -%} {%- assign cat_slug = cat_name | slugify -%} {%- assign _cm = site.data.blog_categories[cat_name] -%} {%- assign _ca = _cm.accent | default: 'blue' -%} {{ cat_name }} {{ cat_count }} {%- endfor %}
