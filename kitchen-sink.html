Typography — In Prose Wrapper

.prose

Prose H2 — 48px top margin, text-3xl

This is a paragraph inside .prose. It gets 1.5em bottom margin instead of 0.75em. The prose wrapper is max-width 720px (--width-lg), centered with auto margins and 12px side padding.

Prose H3 — 48px top margin, text-2xl

Another paragraph. Notice the spacing is more generous inside prose. Bold inline code appears with a background highlight. This is the wrapper used for all tutorial and guide content.

Prose H4 — 32px top margin, muted color

H4 headings inside prose are color-text-muted, providing a visual hierarchy step down from h3.

  • Prose list items get 0.4em bottom margin
  • Lists have 1.5em left padding
  • 8px top margin, 24px bottom margin
  1. Ordered lists get the same treatment
  2. Same spacing, same indentation
  3. Same font size (base)
.prose-report (justified)

This is prose-report text. It uses justified alignment with 1.65 line-height and 28px bottom margin. This is the style used for the childcare report and any future long-form narrative content. The wider spacing and justification create a more formal, publication-like feel compared to standard prose.

A second paragraph to show the 28px gap between paragraphs. Compare this to the standard prose gap above.

This tutorial shows how to retrieve unemployment data from the BLS API using Python.

Step 1: Setup

First, install the required packages and import them:

Import packages 
import requests
import pandas as pd
import json

Making the Request

The BLS API v2 requires an API key. Set up the request parameters:

headers = {'Content-type': 'application/json'}
data = json.dumps({
    "seriesid": ["LNS14000000"],
    "startyear": "2020",
    "endyear": "2026"
})

Response Format

The API returns JSON with nested series data. Here's what the parsed DataFrame looks like:

yearperiodvalue
02026M034.1
12026M024.2
22026M014.0
Full Response Structure

The raw JSON response includes metadata about the request status, series catalog information, and the data array.

{
  "status": "REQUEST_SUCCEEDED",
  "Results": {
    "series": [{
      "seriesID": "LNS14000000",
      "data": [...]
    }]
  }
}
Report pattern: prose-report → blockquote → info-box → highlight

The childcare sector in the United States faces a fundamental structural challenge. Unlike most industries, the cost of providing childcare is dominated by a single factor: labor. Approximately 70-80% of operating costs go to staff wages and benefits, leaving almost no room for productivity gains through technology or scale.

"The economics of childcare are unlike any other industry. You cannot automate caring for a two-year-old." — Center for American Progress, 2025
Key Finding

Cost Burden

The average US family with an infant spends 24% of household income on childcare — more than housing in most metro areas.

$14,760

Average annual cost of center-based infant care, 2024

This cost structure creates a painful trilemma: childcare workers earn poverty-level wages (median $13.71/hr), parents pay unaffordable prices, and providers operate on razor-thin margins. No market adjustment can resolve all three simultaneously without external funding.

International Comparison

Nordic countries resolved this trilemma decades ago through public investment. In Finland, parents pay a maximum of €295/month regardless of care type, while childcare workers earn middle-class wages with full benefits and pension.
